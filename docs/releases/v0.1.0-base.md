# v0.1.0 — CRM Base

**Data:** 2026-05 (retroativa)
**Onda:** 1

## Resumo

Primeira versão funcional da Orca: cadastro de clientes, criação de propostas com itens, status configuráveis e link público compartilhável com cliente final.

## Novidades para o usuário

- 👥 Cadastro de clientes (nome, empresa, e-mail, telefone, endereço)
- 📄 Editor de propostas com itens (descrição, quantidade, valor unitário, total)
- 🎯 Status de propostas configuráveis (rascunho, enviada, aprovada, recusada)
- 🔗 Link público para o cliente visualizar a proposta sem login
- ✍️ Assinatura digital simples (cliente digita nome e aceita)
- 📊 Listagem com filtros por cliente e status

## Mudanças técnicas

### Adicionado
- Tabelas: `clients`, `proposals`, `proposal_items`, `proposal_statuses`, `proposal_signatures`, `proposal_status_history`, `proposal_views`
- RLS por `auth.uid()` em todas as tabelas de tenant
- Trigger de histórico de status (`track_proposal_status_change`)
- RPC `accept_proposal(p_code)` e `sign_proposal(p_code, ...)` para fluxo público

## Métricas e limites afetados

- Sem limites — todos os usuários têm acesso completo.
