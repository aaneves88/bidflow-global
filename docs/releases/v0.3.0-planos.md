# v0.3.0 — Planos & Watermark

**Data:** 2026-06 (retroativa)
**Onda:** 3

## Resumo

Modelo de monetização introduzido: três planos (Free, Pro, Business) com limites por uso, personalização de marca restrita aos pagantes e watermark "Orca" no PDF e link público dos usuários Free.

## Novidades para o usuário

- 🆓 **Free**: 3 propostas no total, com marca d'água Orca, sem personalização
- ⭐ **Pro**: propostas ilimitadas, branding completo (logo + cores)
- 🚀 **Business**: tudo do Pro + remoção total de "powered by Orca"
- 📊 Indicador de uso visível no topo do app
- 🎨 Tela de configuração de marca (logo da empresa, cores primária/secundária/destaque)

## Mudanças técnicas

### Adicionado
- Tabelas `plans` e `user_plans`
- Hook `usePlanLimits` para checagem reativa
- Hook `useBranding` com fallback para defaults Orca em Free
- Watermark diagonal e rodapé "Feito com Orca" no PDF
- Indicador de uso (`UsageIndicator`) no `AppLayout`

### Alterado
- `ProposalView` e `PublicProposal` respeitam plano do dono ao aplicar branding
