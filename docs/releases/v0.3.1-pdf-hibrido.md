# v0.3.1 — PDF Híbrido

**Data:** 2026-06-09
**Onda:** 3.5

## Resumo

Reescrita da geração de PDF para combinar renderização visual de alta fidelidade (html2canvas) com texto nativo selecionável (jsPDF), eliminando os bugs de kerning e sobreposição da versão anterior.

## Novidades para o usuário

- 📄 PDF visualmente idêntico ao link público (mesmas cores, logo, layout)
- 🔍 Tabela de itens com texto selecionável e pesquisável
- 📑 Paginação automática com URL pública no rodapé
- 💧 Marca d'água diagonal "Orca" no plano Free
- 🏷️ Rodapé "Feito com Orca" discreto em todas as páginas (exceto Business)

## Mudanças técnicas

### Adicionado
- Dependência `html2canvas`
- Componente off-screen `PdfHero` que renderiza hero com branding do dono
- Captura via canvas em escala 2x com `useCORS`

### Corrigido
- Bug de kerning em `autoTable` (forçado `font: 'helvetica'`)
- Sobreposição de título e nome da empresa no cabeçalho
- Mensuração incorreta do box de destaque do total
