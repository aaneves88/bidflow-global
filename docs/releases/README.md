## Convenção

Versionamento semântico (semver). Enquanto pré-1.0:
- `0.MINOR.0` → nova onda do roadmap (mudança visível ao usuário)
- `0.MINOR.PATCH` → ajuste menor dentro da mesma onda

A `v1.0.0` marca o lançamento comercial público.
